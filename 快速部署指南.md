## 🚀 第一步：安装Python

### Windows用户

1. 访问：https://www.python.org/downloads/
2. 点击黄色按钮下载Python
3. 双击安装包
4. **重要：勾选 "Add Python to PATH"**
5. 点击"Install Now"
6. 等待安装完成

### Mac用户

打开"终端"，复制粘贴以下命令：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python@3.11
```

---

## 📥 第二步：下载项目

### Windows用户

1. 按 `Win + R`，输入 `cmd`，回车
2. 输入以下命令进入桌面：
   ```bash
   cd %USERPROFILE%\Desktop
   ```
3. 下载项目：
   ```bash
   git clone https://github.com/orgMir123/dou-image-tool.git
   cd dou-image-tool
   ```

### Mac用户

打开终端，输入以下命令：

```bash
cd ~/Desktop
git clone https://github.com/orgMir123/dou-image-tool.git
cd dou-image-tool
```

**注意：** 如果提示"git不是内部或外部命令"，需要先安装Git：
- Windows: https://git-scm.com/download/win
- Mac: `brew install git`

---

## 📦 第三步：安装依赖

### Windows用户

在命令行窗口依次输入：

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### Mac用户

在终端依次输入：

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

**注意：** 这一步可能需要5-10分钟，请耐心等待。

---

## 🎬 第四步：启动项目

### Windows用户

```bash
python web_app.py
```

### Mac用户

```bash
python3 web_app.py
```

看到以下信息表示启动成功：

```
============================================================
                    Web抠图工具
============================================================

访问地址: http://localhost:5001
```

---

## 🌐 第五步：打开浏览器

在浏览器地址栏输入：

```
http://localhost:5001
```

按回车，即可看到工具界面！

## ❓ 常见问题

### Q1: 提示"找不到python命令"

**Windows：** 重新安装Python，记得勾选"Add Python to PATH"
**Mac：** 使用 `python3` 代替 `python`

### Q2: pip安装很慢

使用国内镜像源：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3: 语音合成失败

不用担心！项目已内置密钥，如果遇到问题：
1. 检查网络连接
2. 重启程序试试
3. 联系项目提供者

### Q4: 如何停止服务

在命令行窗口按 `Ctrl + C`

### Q5: 下次如何启动

**Windows：**
```bash
cd %USERPROFILE%\Desktop\dou-image-tool
venv\Scripts\activate
python web_app.py
```

**Mac：**
```bash
cd ~/Desktop/dou-image-tool
source venv/bin/activate
python3 web_app.py
```

## 💡 使用技巧

### 语音合成最佳实践

1. **选择服务商**：推荐"腾讯云TTS（营销女声）"
2. **调整情感**：默认"高兴"，适合带货
3. **语速设置**：默认1.0（正常），可根据需要调整
4. **音量设置**：默认10（正常音量）
5. **长文本**：自动分段合成，无需担心字数限制

### 批量抠图技巧

1. 一次可上传多张图片
2. 支持JPG、PNG格式
3. 图片大小建议不超过5MB
4. 处理完成后可批量下载
